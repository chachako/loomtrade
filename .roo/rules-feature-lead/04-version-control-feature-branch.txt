  feature_branch_merge_to_develop:
    trigger: "When all sub-tasks for feature {context.feature_id} are completed, reviews passed, QA passed, and the feature is deemed ready for integration by Feature-Lead (status in `progress.yaml` is 'ready_for_merge' or similar)."
    priority: 870
    action: |
      <thinking>
      **[Merge Feature Branch to Develop - Feature Lead]**
      My status is `[FEATURE_LEAD: PREPARING_MERGE]`. Feature ID: `{context.feature_id}`.
      Feature '{context.feature_name}' is ready to be merged from `feature/{context.feature_id}` into `develop`.
      I need to:
      1.  Update local `feature/{context.feature_id}` and `develop` branches from remote.
      2.  Attempt to merge `feature/{context.feature_id}` into `develop` using `--no-ff`.
      3.  The outcome (success or conflict) will trigger a subsequent rule (`handle_successful_merge` or `handle_merge_conflicts`).
      </thinking>

      "Preparing to merge feature branch `feature/{context.feature_id}` into `develop` for feature '{context.feature_name}'."

      1.  **Update and switch branches:**
          <execute_command><command>git checkout feature/{context.feature_id}</command></execute_command>
          <!-- Wait -->
          <execute_command><command>git pull origin feature/{context.feature_id}</command></execute_command>
          <!-- Wait -->
          <execute_command><command>git checkout develop</command></execute_command>
          <!-- Wait -->
          <execute_command><command>git pull origin develop</command></execute_command>
          <!-- Wait -->
          My status becomes `[FEATURE_LEAD: MERGING_FEATURE_BRANCH]`. Feature ID: `{context.feature_id}`.
      2.  **Perform Merge:**
          <execute_command>
            <command>git merge --no-ff feature/{context.feature_id} -m "Merge feature {context.feature_id}: {context.feature_name}"</command>
          </execute_command>
          <!-- Wait for merge result. This result will be processed by handle_successful_merge or handle_merge_conflicts. -->
          <!-- Feature-Lead is effectively paused here, awaiting the merge command's outcome. -->
